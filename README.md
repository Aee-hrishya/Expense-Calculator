# Expense-Calculator
A simple expense calculator app in which we can track our expenses. The project is made using React.


✨The concepts used in this app:-
- The useState hook.
- Event listeners.
- Props and components(as these are the building blocks of any React app).

✨Features:
- We can add the expenses and can select the year for those expenses.
- We can even filter the expenses as we want according to the year we select.
- While adding the expenses we can even select the amount and name of the expense and exact date 
  when we made that expense.
